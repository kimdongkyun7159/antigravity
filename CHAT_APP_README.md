# 실시간 채팅 앱 💬

Error Analyzer 프로젝트에 추가된 실시간 웹 채팅 애플리케이션입니다.

## 🚀 주요 기능

### ✨ 실시간 통신
- **Socket.IO 기반**: WebSocket을 사용한 실시간 양방향 통신
- **즉시 메시지 전달**: 서버를 통한 실시간 메시지 브로드캐스트
- **접속자 수 표시**: 현재 온라인 사용자 수 실시간 업데이트
- **입장/퇴장 알림**: 사용자 입장 및 퇴장 시 자동 알림

### 🎨 모던 UI/UX
- **Glassmorphism 디자인**: 투명한 유리 효과의 현대적인 디자인
- **Dark Mode**: 눈에 편안한 다크 테마
- **반응형 레이아웃**: 모바일, 태블릿, 데스크톱 모두 지원
- **부드러운 애니메이션**: 메시지 등장, 버튼 호버 등 세련된 효과

### 💡 사용자 편의
- **닉네임 설정**: 자유롭게 닉네임 지정 가능
- **메시지 타임스탬프**: 각 메시지의 전송 시간 표시
- **자동 스크롤**: 새 메시지 도착 시 자동으로 스크롤
- **Enter 키 전송**: 엔터 키로 간편하게 메시지 전송

## 📦 기술 스택

### Backend
- **Flask-SocketIO 5.6.0**: 실시간 통신 서버
- **Python 3.11**: 서버 사이드 로직
- **Socket.IO**: WebSocket 프로토콜 구현

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: Glassmorphism, 애니메이션, 반응형 디자인
- **Vanilla JavaScript**: 순수 자바스크립트 (프레임워크 없음)
- **Socket.IO Client 4.5.4**: 클라이언트 사이드 실시간 통신

## 🎯 빠른 시작

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

필수 패키지:
- `flask-socketio>=5.3.0` (자동으로 Socket.IO 의존성 설치)

### 2. 서버 실행

```bash
python app.py
```

또는

```bash
python -m antigravity
```

### 3. 채팅 접속

브라우저에서 다음 주소로 접속:

```
http://localhost:5000/chat
```

## 💻 사용 방법

### 1. 채팅방 입장
1. 채팅 페이지 접속
2. 닉네임 입력 (2글자 이상, 20글자 이하)
3. "입장하기" 버튼 클릭

### 2. 메시지 전송
1. 하단 입력창에 메시지 입력
2. 전송 버튼 클릭 또는 Enter 키 입력
3. 모든 접속자에게 실시간으로 메시지 전달

### 3. 채팅방 나가기
1. "나가기" 버튼 클릭
2. 확인 메시지에서 "확인" 선택

## 📡 Socket.IO 이벤트

### 클라이언트 → 서버
- `connect`: 서버 연결
- `disconnect`: 연결 해제
- `join`: 채팅방 입장 (닉네임 전송)
- `leave`: 채팅방 퇴장
- `message`: 메시지 전송

### 서버 → 클라이언트
- `user_joined`: 새 사용자 입장 알림
- `user_left`: 사용자 퇴장 알림
- `user_count`: 현재 접속자 수
- `message`: 메시지 브로드캐스트
- `error`: 에러 메시지

## 🔐 보안 고려사항

### 현재 구현
- 닉네임 길이 제한 (2-20자)
- 메시지 길이 제한 (500자)
- CORS 설정
- Session ID 기반 사용자 관리

### 향후 개선 사항
- 메시지 입력 검증 강화 (XSS 방지)
- Rate limiting (스팸 방지)
- 사용자 인증 시스템
- 채팅 내용 암호화
- 채팅 로그 저장 (옵션)

## 📈 성능

- **동시 접속자**: 수백 명 지원 (Socket.IO 최적화)
- **메시지 지연**: < 100ms (로컬 네트워크)
- **메모리 사용**: 최소화 (메시지 히스토리 미저장)
